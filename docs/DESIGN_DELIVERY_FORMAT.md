# 設定依頼フォーマット出力 + 監査ログ 設計

## 概要

SSMIXパース → エラー検出 → マッピング完了後、
病院/ベンダーに渡す設定依頼フォーマットを自動生成する。
全操作は監査ログに記録する。

## 出力フォーマット

### ①依頼用（全ベンダー共通）
```
| ローカルコード | 項目名称 | CS名 | JLAC10 | JLAC10標準名称 |
|--------------|---------|------|--------|---------------|
| L02715 | IDH1変異解析（骨髄） | 99Z14 | 8C571995104686200 | IDH1(DNA点突然変異解析)_ 骨髄液_リアルタイムPCR法_IDH1 |
```
※依頼側ローカルコードはLプレフィックス（富士通の場合）

### ②結果用（全ベンダー共通）
```
| ローカルコード | 項目名称 | CS名 | JLAC10 | JLAC10標準名称 |
|--------------|---------|------|--------|---------------|
| 0053200 | P-AMY | 99Z14 | 3B175000002327101 | 膵アミラーゼ_ 血清_可視吸光光度法_定量値 |
```

### ③JJ用（富士通専用）
```
| HL7KEY | ICODE | JJCODE | JJNAME |
|--------|-------|--------|--------|
| 0053200 | L | 3B175000002327101 | 膵アミラーゼ_ 血清_可視吸光光度法_定量値 |
```
※ICODE: データなしの場合 'L' を入力（登録エラー防止）
※JJNAME: 128Byte制限、空は登録エラー
※HL7KEY: 富士通細菌検査は特殊プレフィックス:
  - 細菌結果: "OTKKINF" + 項目コード
  - 塗抹: "ER02TMTCD" + 菌コード
  - 同定: "ER02DTKCD" + 菌コード
  - 抗酸菌塗抹: "ER03TMTCD" + 菌コード
  - 抗酸菌同定: "ER03DTKCD" + 菌コード
  - 抗菌薬: "KJYKCD" + 菌コード

### ④NEC用（細菌検査結果）
NEC指定フォーマット（別途定義）

### ⑤SSI用（細菌検査）
SSI用細菌検査Lookup形式（別途定義）

## JLAC10標準名称の生成ルール

```
{分析物名}_ {材料名}_{測定法名}_{結果識別名}
例: 膵アミラーゼ_ 血清_可視吸光光度法_定量値
```
jlac10_lookup.json の各パートから名称を取得して連結。

## 監査ログ

### 記録する操作
| 操作 | 記録内容 |
|------|---------|
| SSMIXパース | ファイル名、メッセージ数、エラー数、ベンダー |
| エラー検出 | エラー種別、対象項目、ローカルコード |
| マッピング | 誰が、どのコードを、何に変更したか |
| SOP違反検出 | 違反ルール、対象コード |
| フォーマット出力 | 出力先（病院/ベンダー）、出力件数 |
| メール送信 | 宛先、件名、送信日時 |
| DB還元 | 追加件数、病院名 |

### ログ構造
```json
{
  "audit_log": [
    {
      "id": "A00001",
      "timestamp": "2026-04-21T10:00:00",
      "user": "umetani",
      "action": "mapping",
      "issue": "#4123",
      "hospital": "H100",
      "detail": {
        "local_code": "0053200",
        "item_name": "P-AMY",
        "from_jlac10": "",
        "to_jlac10": "3B175000002327101",
        "standard_name": "膵アミラーゼ_ 血清_可視吸光光度法_定量値"
      }
    }
  ]
}
```

## 実装計画

### Phase 1: フォーマット出力（Python CLI）
- `srl-scraper export-delivery` コマンド
- 依頼用/結果用/JJ用 のExcel出力
- ベンダー自動判定（CS名から）

### Phase 2: Pages UI
- SN Error アクションタブにフォーマット出力ボタン
- マッピング完了 → ワンクリックで設定依頼Excel生成

### Phase 3: 監査ログ
- 全操作の自動記録
- Pages: LocalStorage
- CLI: audit_log.json
- 検索・フィルタ機能
